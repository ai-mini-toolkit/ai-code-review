# AI 智能代码审查系统

AI Code Review System - 基于 AI 的智能代码审查工具

## 项目结构

```
ai-code-review/
├── frontend/     # Vue + Vue-Vben-Admin 前端
└── backend/      # Spring Boot 后端
```

## 技术栈

**后端：** Java + Spring Boot + JPA
**前端：** Vue + Vue-Vben-Admin + Element Plus

## 开发

使用 BMAD Method 进行开发。

详见各子目录的 README。
